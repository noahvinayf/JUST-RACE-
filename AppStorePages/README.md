# Pages App Store - JUST RACE

Ce dossier contient les pages publiques a mettre en ligne pour App Store Connect.

Fichiers a importer dans le repo GitHub `just-race` :

- `index.html`
- `privacy.html`
- `support.html`
- `styles.css`

Une fois GitHub Pages active, les URL seront :

```text
https://TON-PSEUDO.github.io/just-race/privacy.html
https://TON-PSEUDO.github.io/just-race/support.html
```

Ces deux liens sont ceux a mettre dans App Store Connect :

- Privacy Policy URL : `https://TON-PSEUDO.github.io/just-race/privacy.html`
- Support URL : `https://TON-PSEUDO.github.io/just-race/support.html`
