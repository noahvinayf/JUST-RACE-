Support - JUST RACE

Pour contacter le support de JUST RACE, signaler un bug, demander de l'aide ou proposer une amélioration, envoyez un email à :

noah.vinay.fr@gmail.com

Merci d'indiquer si possible :
- le modèle de votre iPhone ;
- la version iOS ;
- la version de JUST RACE ;
- une description claire du problème ;
- les étapes pour reproduire le bug ;
- une capture d'écran ou une vidéo si possible.

JUST RACE est une application de télémétrie utilisant les capteurs de l'iPhone pour mesurer la vitesse, la distance, les forces G, l'inclinaison, le roulis et les données de session.

Les mesures peuvent varier selon le modèle d'iPhone, la qualité du signal GPS, les conditions d'utilisation et la position de l'iPhone dans le véhicule.

JUST RACE doit être utilisée de manière responsable, principalement sur circuit, terrain privé ou zone sécurisée. Ne manipulez pas votre iPhone en conduisant.
